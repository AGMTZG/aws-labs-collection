# CloudCart – Project README

## 1. Project Description

CloudCart is a sample static website created for learning and practicing **AWS S3** static website hosting.

## 2. Folder Structure

```bash

assets/
├── website-assets/
├── logs/
├── backups/
└── important-docs/

```

## 3. Learning Objectives

- Understand S3 static website hosting
- Practice bucket policies and permissions
- Work with logs and backups
- Apply lifecycle rules

## 4. Suggested Exercises

1. Upload website files to S3
2. Enable static website hosting
3. Configure bucket policy
4. Enable access logs
5. Apply lifecycle rules
6. Restore from backups

## 5. Disclaimer

**This project is for educational purposes only.**  
Do not use it as-is in production environments.
